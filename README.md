# Payroll Calculator - React Native

React Native payroll calculator application built with Expo.

## Features

- Calculate regular pay (up to 40 hours)
- Calculate overtime pay (hours over 40 at 1.5x rate)
- Calculate total pay and tax
- About screen with student info

## Important

**Tax is calculated on total pay** (base pay + overtime pay), not just base pay.

## Test Values

Example input:
- Hours: `50`
- Rate: `20`
- Tax: `10`

Expected:
- Pay: $800, Overtime: $300, Total: $1,100, Tax: $110
